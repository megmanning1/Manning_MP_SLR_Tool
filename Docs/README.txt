The documents in this folder contain:

1) SLR Inundation Methods - NOAA 2017

Description: the methodology for mapping desired sea level rise onto a Digital Elevation Model (DEM).

National Oceanic and Atmospheric Administration. (2017, January). Sea level rise inundation methods: Technical documentation. NOAA Office for Coastal Management. Retrieved November 21, 2024, from https://coast.noaa.gov/data/digitalcoast/pdf/slr-inundation-methods.


2) Regional Coastal Resilience Assessment 

Description: this outlines the data and methodology for creating a coastal resilience assessment using the Coastal Resilience and Evaluation and Siting Tool (CREST). 

Accessed through: https://resilientcoasts.org/Home 


3) Climate Resilience Data 

Description: this slide deck provides other relevant climate resilience data available for the state of North Carolina.
